## How to Repo to Github:

1. cd ~/Name-of-Folder
2. touch README.md 
* git status (optional)
3. git commit -m "comment"
4. copy git remote line and paste on terminal
5. git push
6. git push --set-upstream origin master
7. open README.md

## How to Update File info to Github Repo:

1. cd ~/Documents
2. ls
3. git init
* git status (optional)
4. git add README.md 
* git status (optional)
5. git push
6. git commit -m "comment"
